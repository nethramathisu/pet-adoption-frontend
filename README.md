# Pet Adoption Portal

A full-stack pet adoption website where users can browse pets and apply for adoption online.

## Features

* User login and registration
* Browse available pets
* View pet details
* Apply for pet adoption
* Foster pets temporarily
* Shelter can add, edit, and remove pets
* Shelter can manage adoption and foster requests

## User Roles

### Adopter

* Browse pets
* Apply for adoption
* Track application status

### Foster

* Apply to foster pets temporarily
* Manage foster requests

### Shelter

* Add new pets
* Update pet details
* Remove pets
* Manage adoption applications

## Tech Stack

### Frontend

* React
* Tailwind css
* TypeScript
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication


## Deployment

* Frontend: Netlify
* Backend: Render
* Database: MongoDB Atlas
